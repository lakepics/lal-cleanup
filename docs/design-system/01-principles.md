# 01 Principles

Status: approved
Source sections: 1, 2

## Role
- Translate brand strategy into reusable interface rules.
- Support editorial storytelling and practical hospitality UX.
- Balance warmth, refinement, clarity, and restraint.

## Objectives
- Consistency, scalability, hierarchy clarity, accessibility, maintainability.
- Preserve property-specific character.

## Source-of-Truth Layers
1. Foundations
2. Elements
3. Components
4. Patterns
5. Templates
6. Implementation rules

## Rule Precedence
1. Latest approved brand positioning
2. Verified property facts
3. Approved design-system tokens and component rules
4. Current site implementation
5. Legacy design or copy
